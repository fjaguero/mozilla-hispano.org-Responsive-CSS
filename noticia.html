La versión para Android disponible en tabletas

29 agosto, 2012 10:00 por deimidis

actualizaciones android firefox firefox mobile

Los mismos cambios que hicieron que Firefox para dispositivos Android fuera súper rápido están ahora disponibles para tabletas. Al comenzar a utilizarlo te darás cuenta de que todo es más rápido, desde el inicio hasta la carga de páginas, hacer zoom y el desplazamiento es mucho más fluido, además de una notable mejoría en el rendimento de las aplicaciones Web.

Firefox para Android hace más fácil mantener tus sitios favoritos a mano con una página de inicio personalizada, con un estilo más refinado y ligero. Además esta actualización incluye la pantalla increíble que contiene tu historial de navegación, tus marcadores, contraseñas y datos almacenados de formularios. Puedes incluso importar todas estas configuraciones por medio de Firefox Sync y mejorar tu experiencia personalizada en otros dispositivos.

Firefox para Android permite usar Flash para tabletas que tengan la versión Honeycomb del sistema operativo, con lo que puedes interactuar fácilmente con mas vídeos Web y juegos.
Para facilitar la navegación entre la versión móvil y la tableta, Firefox te permite cambiar entre la versión de escritorio y la versión móvil de una página web con tan solo seleccionar la opción “Pedir sitio para Escritorio” desde el menú del Navegador, además de otras características que mejoran la estabilidad y el rendimiento, así como te facilitan tareas como “Buscar en la página” y “Seleccionar Texto”.

Basado en la misma plataforma tecnológica que la versión de escritorio, tiene los níveles más altos de privacidad y seguridad para la navegación móvil. Firefox respeta tu privacidad y protege tu seguridad con características como “No quiero ser rastreado”, el uso de una contraseña maestra, HTTP Strict Transport Security y más.

Las nuevas funcionalidades HTML5 de Firefox para Android permiten a los desarrolladores crear aplicaciones y sitios web basados en HTML5, CSS y otros estándares web abiertos. Como defensor de la Web abierta, Mozilla expone las nuevas APIs Web y las presenta para ser estandarizadas con el fin de que la Web avance como plataforma. Entre los estándares que Mozilla ha ayudado a construir están la API para cámaras, API para vibración, API para conexiones móviles, API para el estado de la batería, API para la orientación de la pantalla y la API para geolocalización.

Te invitamos a jugar BrowserQuest HTML5 de Mozilla, un juego de rol multi jugador que puedes disfrutar entre diferentes dispositivos de escritorio y móviles, para que veas qué tan poderosa, rápida y divertida puede ser la Web.
